시간표 작성 프로젝트를 작성한다.
시간표 내용은 구글 캘린더와 같은 화면으로 조회할 수 있어야 한다.
교수들은 교수별로 본인의 시간표를 조회할 수 있고 연가를 신청하면 연가 기간의 일정을 원하는 일정으로 수정할 수 있어야한다.
일정 수정시 해당시간에 과목이 배정되어 있으면 다른 일정으로 입력하도록 해야한다.
시간표는 기간을 설정하여 해당 기간의 시간표를 엑셀로 다운받을 수 있어야한다.
기본 자료는 엑셀파일을 선택하여 업로드하여 sqlite에 저장하여 사용한다.
'시간표.xlsx' 파일은 날짜, 교시, 과목, 교수로 이루어져 있다. 
'일정.xlsx' 파일은 교시, 시작시간, 종료시간으로 이루어져 있다.
'교수.xlsx' 파일은 교수명, 담당과목으로 이루어져 있고 담당과목은 쉼표로 구분되어 있다.
'과목.xlsx' 파일은 과목명, 배정시간으로 이루어져 있다.
시간표.xlsx 파일이 업로드 되면 해당 기간의 DB에 있던 내용은 모두 삭제되고 업로드된 내용으로 다시 작성되어야한다.
백엔드와 프론트엔드로 나누어 작성한다. 
프론트엔드는 리액트 프로젝트로 진행하고 tailwindcss를 사용하며 DB는 sqlite를 사용한다. 
프론트엔드는 기능별로 컴포넌트로 분리해서 작성해주고 라우팅이 되도록 설정한다.
캘린더 UI는 react-big-calendar를 사용한다.
이프로젝트의 PRD를 작성해서 prd.md 파일로 저장해줘.


prd.md 파일을 참고하여 프로젝트 환경설정 파일을 setup.md 파일로 작성해줘.

프론트엔드 메인 페이지를 랜딩페이지로 작성해줘. 전체 화면디자인을 최신 트렌드를 반영하여 tailwindcss를 활용하여 변경해주고 반응형으로 작성해줘.
랜딩페이지의 시작하기를 누르면 prd.md 파일을 참고하여 메뉴를 구성해줘. 각 기능은 컴포넌트로 분리하고 라우팅이 되도록 작성해줘.

데이터 관리에서 prd.md 파일을 참고해서 각각의 엑셀 파일을 업로드해서 sqlite db에 반영할 수 있도록 작성해줘.

시간표 전체 조회 메뉴를 추가하고 구글 캘린더 형식으로 화면에 조회할 수 있도록 작성해줘.
